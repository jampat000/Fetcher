This release focuses on production hardening and stability following the 3.0.1 rollout.

Key improvements:
- Refiner state accuracy (off vs enabled vs ready)
- Async UI sync fixes for Refiner settings
- Folder picker isolation using subprocess (prevents hangs and crashes)
- Graceful fallback when folder picker is unavailable
- Standardized error handling and messaging

This release contains no new features — only reliability improvements.
